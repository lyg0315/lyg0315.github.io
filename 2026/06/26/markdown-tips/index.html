## 一句话总结

| 技巧 | 场景 |
| --- | --- |
| 语言标注 | 代码块第一行写语言标识，获得语法高亮 |
| Diff 语法 | 展示代码变更时用 `diff` 语言标识 |
| 表格对齐 | `:---` 左、`:---:` 中、`---:` 右 |
| `<details>` | 折叠长内容，保持页面清爽 |
| 任务列表 | 做 checklist、发布前自检 |
| 引用提示块 | 用 `> 💡` 等 Emoji 做视觉分层 |
| Mermaid | 流程图、时序图直接写在 Markdown 里 |
| HTML 嵌入 | 控制图片尺寸、并排布局等高级排版 |
| 链接引用式 | 长文档中 URL 集中管理 |

Markdown 的精髓在于：**用最少的形式干扰，表达最清晰的内容结构**。掌握这些技巧，你的文档会从"能看"变成"好看"。
